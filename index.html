Print Production · Spec Sheet

Storefront & Apparel Release

Three print-ready assets built from the existing Singer Center mark — T-shirt front, entry door decal, and storefront window decal. Vector SVG, sized to real production dimensions, set in Poppins Light.

Full color · emblem on a white disc, works on any garment color

Single-color charcoal · economy screenprint, light garments only

01 — APPAREL

T-Shirt Front Print

Standard centered-chest placement, sized for a true adult unisex tee without crowding a standard 12″×16″ press area.

Design size9.41″ W × 7.25″ H
File canvas10.01″ × 7.85″ (incl. 0.3″ margin)
PlacementCentered horizontally; top of emblem ~3″ below the collar seam
Scales to7″–11″ wide (youth to XXL) — keep width:height locked at 1.30:1
Color builds2-color (charcoal + teal) on a white disc, or 1-color charcoal for economy screenprint

The emblem sits on a solid white disc so it holds its contrast on any garment color, not just white/light ones — no separate dark-garment version needed. The wordmark itself is still plain teal with no backing, so on very dark garments it will read with less contrast than the emblem above it.

02 — ENTRY DOOR

Door Decal

Right-hand door as viewed from the sidewalk, sized to the top 38″ of the door only — the zone that naturally lands at eye level for someone approaching the entrance. Emblem and wordmark only, no website or handle — kept as a clean brand moment.

Artboard30.5″ × 38″ (top of door only)
Design block24.91″ W × 19.18″ H
PlacementCentered on the panel, both axes
MaterialCut vinyl, applied to exterior glass (see tint note below)

This glass has reflective privacy tint: mount on the exterior face so the tint sits behind the decal, not in front of it. The emblem's white backing disc already gives it strong contrast against both a bright daytime reflection and a darker interior at night, so no separate reversed colorway is needed here.

Full color · 48″ × 50″ top pane

03 — STOREFRONT WINDOW

Window Decal — Service List

Left-hand window, top pane. The logo lives on the door, so this pane carries the supporting information instead: a microphone, piano, theater masks, a REC indicator, and a camera for Voice, Piano, Coaching, Production, and Photography, each on its own white-backed ring badge so it holds up the same way the emblem does. A staggered 3-over-2 layout uses the pane's height as well as its width, and reads faster from a moving car than a line of type would at the same scale.

Artboard48″ × 50″ (top pane only)
Design block~40.2″ W × 19.7″ H
PlacementCentered on the pane, both axes
MaterialCut vinyl, applied to exterior glass (same tint consideration as the door)

Bottom 30″ pane intentionally left bare — keeps sightlines into the studio clear.

Palette

Charcoal (ink)#3E404C · C18 M16 Y0 K70
Teal (accent)#1E698D · C79 M26 Y0 K45
White (emblem backing)#FFFFFF · C0 M0 Y0 K0

Typography

Poppins Light, all caps, +3% tracking — matches the existing wordmark exactly.

Text is rendered as vector outlines in every file, so no font needs to travel with the artwork or be installed at the print shop.

Vendor notes

Files are true vector SVG at 1:1 print scale — open directly in Illustrator, CorelDRAW, or sign-cutting software (SignCut, FlexiSign, etc).

CMYK values above are approximate screen conversions for reference; ask your vendor to match ink/vinyl to the hex codes directly for the truest result.

Window service icons are from Tabler Icons (MIT) and Google Material Symbols (Apache-2.0), recolored to match — both licenses are free to use commercially with no attribution required. The Production icon is set directly in Poppins rather than a stock glyph.

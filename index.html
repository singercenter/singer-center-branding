02 — ENTRY DOOR

Door Decal

Right-hand door as viewed from the sidewalk, sized to the top 38″ of the door only — the zone that naturally lands at eye level for someone approaching the entrance.

Artboard30.5″ × 38″ (top of door only)
Design block24.91″ W, 19.18–21.75″ H by variant
PlacementCentered on the panel, both axes
MaterialCut vinyl, applied to exterior glass (see tint note below)

Three content variants included since you're still weighing this one: logo only, logo + thesingercenter.com, and logo + website + @thesingercenter. Hours were deliberately left off — they change too often for a permanent decal.

This glass has reflective privacy tint: mount on the exterior face so the tint sits behind the decal, not in front of it. An interior mount would view the decal through the tinted layer, which darkens colors and can wash it out against the daytime mirror effect — if interior mounting is unavoidable, use the reversed-cream build instead of full-color.

03 — STOREFRONT WINDOW

Window Decal — Service List

Left-hand window, top pane. The logo lives on the door, so this pane carries the supporting information instead: a microphone, piano, theater masks, a REC indicator, and a camera for Voice, Piano, Coaching, Production, and Photography, set inside the emblem's own ring-and-accent-dot treatment so they still read as one family. A staggered 3-over-2 layout uses the pane's height as well as its width, and reads faster from a moving car than a line of type would at the same scale.

Artboard48″ × 50″ (top pane only)
Design block~40.2″ W × 19.7″ H
PlacementCentered on the pane, both axes
MaterialCut vinyl, applied to exterior glass (same tint consideration as the door)

Bottom 30″ pane intentionally left bare — keeps sightlines into the studio clear.

Typography

Poppins Light, all caps, +3% tracking — matches the existing wordmark exactly.

Text is rendered as vector outlines in every file, so no font needs to travel with the artwork or be installed at the print shop.

Vendor notes

Files are true vector SVG at 1:1 print scale — open directly in Illustrator, CorelDRAW, or sign-cutting software (SignCut, FlexiSign, etc).

CMYK values above are approximate screen conversions for reference; ask your vendor to match ink/vinyl to the hex codes directly for the truest result.

Window service icons are from Tabler Icons (MIT) and Google Material Symbols (Apache-2.0), recolored to match — both licenses are free to use commercially with no attribution required. The Production icon is set directly in Poppins rather than a stock glyph.
